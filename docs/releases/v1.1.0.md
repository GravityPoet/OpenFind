## macOS First Launch / macOS 首次启动

### English 🌐

1. Download `OpenFind.zip` using the recommended link above.
2. Double-click the ZIP to extract `OpenFind.app`.
3. Drag `OpenFind.app` into **Applications**.
4. Open **Finder → Applications**, then double-click **OpenFind**.
5. If macOS blocks the first launch:
   - Close the security alert and open **System Settings → Privacy & Security**.
   - Scroll to the **Security** section and click **Open Anyway** next to OpenFind.
   - Confirm **Open** in the final dialog. Enter your password or use Touch ID if macOS asks.

**Open Anyway** appears only after macOS has blocked one launch attempt. If
you do not see it, return to **Applications**, try opening OpenFind once, and
then check **Privacy & Security** again. This approval is normally needed only
for the first launch.

### 中文 🇨🇳

1. 使用上方推荐链接下载 `OpenFind.zip`。
2. 双击 ZIP，解压得到 `OpenFind.app`。
3. 把 `OpenFind.app` 拖入 **“应用程序”**。
4. 打开 **Finder → 应用程序**，双击 **OpenFind**。
5. 如果 macOS 拦截首次启动：
   - 关闭安全提示，打开 **“系统设置 → 隐私与安全性”**。
   - 向下滚动到 **“安全性”** 区域，点击 OpenFind 旁边的 **“仍要打开”**。
   - 在最后的确认窗口中点击 **“打开”**；如果系统要求，请输入密码或使用 Touch ID。

**“仍要打开”只会在 OpenFind 被拦截过一次后出现。** 如果没有看到该按钮，
请先回到 **“应用程序”** 双击 OpenFind 一次，再重新查看 **“隐私与安全性”**。
这个确认通常只需在首次启动时操作一次。
