# OpenFind v1.1.0

OpenFind v1.1.0 is the first formal public release.

## Highlights

- A lightweight first-run guide explains all five tools and their current
  shortcuts without turning setup into a wizard.
- Search and status controls automatically compress at the minimum window
  size while preserving tooltips and accessibility labels.
- Compact, Default, and Large interface sizes consistently scale text,
  controls, and spacing across OpenFind windows.
- Approved visual baselines now guard the first-run experience against
  clipping, overlap, and accidental layout drift.
- The release includes a Universal macOS 14+ app, signed Sparkle updates,
  customer-verifiable checksums, three real product screenshots, and a
  60-second demo.

## Installation

Download `OpenFind.zip` and `OpenFind.zip.sha256` from the GitHub Release, put
them in the same directory, and run:

```bash
shasum -a 256 -c OpenFind.zip.sha256
```

Move `OpenFind.app` to `/Applications`. This release uses OpenFind's pinned
self-signed customer certificate and is not Apple-notarized. If macOS blocks
the first launch, use **System Settings → Privacy & Security → Open Anyway**
once.

## Compatibility

- macOS 14 or later
- Apple silicon and Intel
- Bundle identifier: `com.openfind.app`
- Version: `1.1.0`
- Build: `1001000`
