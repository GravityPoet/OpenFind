## When to Request a Commercial License

Request a commercial license if you want to:

- embed OpenFind code in a closed-source product;
- distribute a modified version without releasing the complete corresponding
  source under the AGPL;
- provide a hosted or managed service based on modified OpenFind code without
  offering users the corresponding source required by AGPL section 13;
- ship private enterprise extensions as part of a combined OpenFind-based work;
- redistribute OpenFind under OEM, white-label, or other proprietary terms.

## AGPL Path

You do not need a commercial license if you comply with `AGPL-3.0-only`.

At a high level, AGPL users must preserve notices, provide the license text, and
release covered modifications and combined covered works under the AGPL. If a
modified version supports remote network interaction, the operator must also
offer the corresponding source to users interacting with that modified version
over a network.

The full and controlling terms are in [LICENSE](LICENSE). If this summary
conflicts with the AGPL, the AGPL controls for AGPL-covered code.
